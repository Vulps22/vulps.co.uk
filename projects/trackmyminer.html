TrackMyMiner

A Minecraft Plugin for Fair Play

The Problem of X-Ray Cheating

TrackMyMiner is a powerful tool I developed to address the persistent issue of "x-ray cheaters" in Minecraft. The idea came from a server owner who needed a trustworthy anti-cheat solution after a previous plugin's developer proved to be unreliable.

I designed this plugin to help server administrators catch suspicious players who use resource packs or mods to locate valuable ores unfairly. When a player mines an excessive number of ores in a short time, the plugin sends a discreet notification to admins, alerting them to potential foul play.

Development & Features

Built entirely in Java and integrated with the Spigot API, this project was a fantastic learning experience. The process required a deep dive into Spigot's documentation to implement robust features like:

  • Statistical Ore Tracking: Monitors mining rates per player.
  • Admin Alerts: Configurable notifications for suspicious activity.
  • Inventory Snooping: Allows admins to discreetly view a player's inventory to verify findings.

The plugin's usage and impact can be tracked live on bStats. As the developer, I am committed to its ongoing maintenance and compatibility with future Minecraft and Spigot releases.
